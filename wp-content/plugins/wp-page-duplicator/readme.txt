== Description ==

Easily duplicate any page or post on your site. Supports taxonomies, custom fields and custom post types.
== Installation ==

1. Download plugin zip file to your computer and extract in a folder
1. Upload `page-duplicator` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
